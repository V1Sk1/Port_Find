Первоначальная цель проекта: Создать графическое приложение на GTК, интерфейс которого
будет состоять из двух кнопок (запуска и отмены поиска), ряда индикаторов с названиями портов.
При нажатии на кнопку запуск будет производится отправка данных на каждый из портов. При ответе 
Микроконтроллером на полученные сообщения, индикатор будет загораться зеленым. 

P.S: Проект пока не закончен.
